# PDFs, Word Docs, Spreadsheets, and CSVs

If you have ever stared at a 47-page PDF and thought, "Absolutely not," this lesson is for you.

One of the most useful things ChatGPT can do is help you work with real files, not just blank text boxes. That means reports, meeting notes, Word documents, spreadsheets, and those glorified text tables we call CSVs.

## Why this matters

Beginners often treat ChatGPT like a chatbot only. Ask a question, get an answer, move on.

But a lot of real work lives inside files. Contracts are in PDFs. Notes are in Word docs. Budgets are in spreadsheets. Customer exports are in CSVs. If you do not know how to use those files well, you end up copy-pasting random chunks and hoping for the best, which is a bit like programming your VCR by guessing.

Using files properly helps ChatGPT give more accurate, grounded, useful answers.

## The idea in plain English

Different file types give ChatGPT different kinds of information:

| File type | Best for | What to ask |
|---|---|---|
| PDF | Reports, manuals, forms, scanned documents | Summarize, extract key points, explain sections |
| Word doc | Drafts, letters, policies, notes | Rewrite, edit, shorten, improve tone |
| Spreadsheet | Tables, budgets, trackers | Find patterns, compare values, summarize trends |
| CSV | Raw exported data | Clean up columns, categorize rows, analyze data |

The big idea is simple: tell ChatGPT both **what the file is** and **what you want done with it**.

Do not just upload a spreadsheet and say, "Help." That is technically a prompt, but not a useful one.

A better prompt looks like this:

## Simple example

**Prompt:**

"I uploaded a CSV of customer support tickets. Please:
1. group the tickets into common issue categories,
2. identify the top 5 complaint themes,
3. suggest 3 improvements the company should make,
4. keep the explanation simple and use bullet points."

That gives ChatGPT a job, a structure, and a clear result.

Here is another:

**Prompt:**

"I uploaded a PDF report. Summarize it for a busy manager in plain English. Include the main findings, risks, and recommended next steps. Keep it under 200 words."

Much better than, "What is in this?"

## Quick tips

- Say what kind of file you uploaded.
- Give a specific task, summarize, extract, compare, rewrite, analyze.
- Say who the answer is for, such as you, your boss, a client, or a beginner.
- Ask for a format, bullets, table, short summary, action items.
- With spreadsheets and CSVs, name the columns if they matter.
- If the result seems off, ask ChatGPT what it used from the file and what assumptions it made.
- For scanned PDFs, results depend on how readable the text is.

## Quick summary

PDFs, Word docs, spreadsheets, and CSVs let ChatGPT work with your actual documents, not just your guesses about them. The trick is not the upload alone. The trick is pairing the file with a clear instruction. When you tell ChatGPT what the file is, what you want, and how to format the answer, the results get much more useful, much faster.
